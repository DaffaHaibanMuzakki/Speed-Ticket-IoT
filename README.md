# Speed-Ticket-IoT
Project ini merupakan tugas dalam hal presentasi mata kuliah sistem sensor dimana melibatkan sensor kecepatan sebagai sensor utama. Cara kerja dari sistem ini 
melibatkan dua ultrasonik sebagai dua titik yang akan dihitung kecepatannya dengan waktu yang dibutuhkan sebuah benda dari titik A sampai ke titik B. Hardware yang digunakan
adalah NodeMcu dengan mikroprosessor ESP8266 dan dua ultrasonik. Software yang digunakan terdiri dari Node.js,ejs,Arduino IDE,MQTT,dan MongoDB.
# Fitur 
1. Menampilkan data kecepatan yang terdeteksi secara real-time dengan MQTT
2. Menampilkan denda jika benda melebihi kecepatan tertentu
